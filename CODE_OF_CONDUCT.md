# Code of Conduct

We are committed to fostering a welcoming, respectful, and inclusive environment for everyone participating in the TREN project.

## Expected Behavior

Contributors are expected to:

- Treat others with respect, professionalism, and courtesy.
- Provide constructive feedback and engage in respectful discussions.
- Welcome different perspectives and experiences.
- Focus on improving the project rather than criticizing individuals.
- Respect the time and effort of other contributors during discussions and code review.

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, discrimination, or intimidation of any kind.
- Personal attacks, insults, or abusive language.
- Publishing another person's private information without permission.
- Deliberately disruptive or unprofessional conduct.
- Any behavior that creates an unwelcoming environment for contributors.

## Reporting Issues

If you experience or observe behavior that violates this Code of Conduct, please contact the project maintainer at **bostanberkay@outlook.com**.

All reports will be handled respectfully and, where possible, confidentially.

## Scope

This Code of Conduct applies to all project spaces, including GitHub issues, pull requests, discussions, and other communication channels related to TREN.

By participating in this project, you agree to follow these guidelines.
